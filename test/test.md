# tslide

Terminal SlideDeck.
  (for back end devs)

---

Controls: 
  * Left, Right: change slide.
  * Ctrl-C     : exit

---

# Usage

```
tslide README.markdown
```

each slide is a section of a markdown document.

"sections" are split at lines that start with `#`
